# Definition
    It is a quiz generator. I can upload files (pdf, pptx), AI read those files, generate questions, let user to answer, grade, add it to history allow user to review it and redo the same questions. 

    - Question : generated from user upload files. porperty : question, answer.
    - Quiz : contain many questions. Generate from user upload files. porperty : number and difficulty. 
    - Session : contain multi quizs. all related to user upload files.
    - Material : user upload files. porperty : file name, file type, file size, upload date.
    - Sessions history : all sessions that user have done. porperty : session name, date, score, weak section.
    

# feature
    - drag or click to upload file
    - user can re do the same Quiz again 
    - Quiz can choose how many questions , amd defficulty Level.
    - UI for user to answer the question, and submit answer. 
    - grade user answer.
